＜使用例１＞
●Angular CLIでプロジェクトを生成して実行する
  ＜コマンド＞
  ng new P001-basic      # WEBアプリ（プロジェクト）を生成 .............（1）
  cd P001-basic          # WEBアプリ（プロジェクト）フォルダーに移動 ...（2）
  ng serve --aot --open  # WEBアプリ（プロジェクト）を実行 .............（3）


＜使用例２＞
●開発サーバーで新しい基本的なAngularプロジェクトを作成、構築、および提供する。

①新しいワークスペースの親ディレクトリに移動する。

②次のコマンドを使用する。
  ＜コマンド＞
  ng new my-first-project
  cd my-first-project
  ng serve

ng new my-first-projectを実行すると、my-first-projectという名前の新しいフォルダーが
作業中のディレクトリに自動的に作成される。 このフォルダの中でWEBアプリファイルを編集することになる。

③手動にて、ブラウザで http://localhost:4200/ を開き、新しいアプリケーションが実行されるのを確認します。
 ng serve コマンドを使用してアプリケーションを構築し、それをローカルに配信すると、
 ソースファイルのいずれかを変更すると、サーバーによって自動的にアプリケーションが再構築され、
 ページがリロードされます。
